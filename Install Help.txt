10 GVH characters modded into Vampire Survivors
Requirements (for all characters to work): Base Game, Tides of the Foscari, Ode to Castlevania
Other Requirements (for modding): MelonLoader (install into the base VS directory, where the application itself is, you ***will*** need to launch the game at least twice for it to install proper), Bloodlines mod (found here: https://github.com/takacomic/bloodlines/releases/tag/0.3.7 , place ***only the bloodline.dll file into the Mods folder after MelonLoader adds it.*** After, unzip the character pack ***but not the zip files inside of it,*** place those files into the added Bloodlines folder ***under UserData folder, has a data folder and a config file.*** Bloodlines will unzip the files for you.)
